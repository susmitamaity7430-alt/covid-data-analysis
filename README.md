🦠 COVID-19 Data Analysis Project
📌 Overview
This project focuses on analyzing COVID-19 data to uncover meaningful insights and trends. The dataset was collected from Kaggle and processed using Python data analysis libraries. The goal is to understand the spread, impact, and patterns of COVID-19 through data visualization and analysis.
📂 Dataset
Source: Kaggle
Contains information such as:
Confirmed cases
Death cases
Recovered cases
Country/Region-wise data
Date-wise records
🛠️ Tools & Technologies Used
Python 🐍
Pandas 📊 (Data cleaning & manipulation)
NumPy 🔢 (Numerical operations)
Matplotlib 📉 (Basic visualization)
Seaborn 🎨 (Advanced visualization)
Jupyter Notebook 📓
🔍 Data Cleaning & Preprocessing
Checked missing values using isnull()
Removed/handled null values
Filtered relevant data
Used value_counts() for categorical insights
Renamed columns for better understanding
Converted data types where necessary
📊 Analysis Performed
Some key questions analyzed in this project:
Total confirmed, deaths, and recovered cases worldwide
Country-wise COVID-19 impact
Top affected countries
Trend of cases over time
Comparison between confirmed, deaths, and recovered cases
Daily growth analysis
📈 Visualizations
Line charts for trend analysis
Bar charts for country-wise comparison
Heatmaps for correlation
Distribution plots
💡 Key Insights
Identified the most affected countries
Observed growth patterns of COVID-19 cases
Compared recovery vs death rates
Found trends in daily case increases
